ЧТО ДЕЛАЕТ ПРОГРАММА
--------------------------------------------
Программа помогает вести список учебных
заданий. Можно добавлять задачи по предметам,
ставить приоритет, указывать дедлайн,
отмечать выполненные и смотреть статистику.
Все данные сохраняются и не пропадают
после закрытия программы.
 
 
КАК ЗАПУСТИТЬ
--------------------------------------------
 
Вариант 1 - готовый exe (проще всего):
  1. Открыть папку с файлом
  2. Дважды кликнуть на "Курсовая ОП Горячева.exe"

Вариант 2 - через Python:
  1. Открыть командную строку в папке с файлом
  2. Написать команду:
     python "Курсовая ОП Горячева.py"
  3. Нажать Enter
 
 
ВАЖНО ПРО ФАЙЛ tasks.db
--------------------------------------------
При первом запуске в той же папке появится
файл tasks.db - это база данных с задачами.
 
  ! Не удалять этот файл !
 
Если его удалить - все задачи сотрутся.
Программа создаст новый пустой файл,
но старые данные не вернуть.
 
exe и tasks.db должны быть в одной папке.
 
 
КАК ПОЛЬЗОВАТЬСЯ
--------------------------------------------
После запуска появится меню:
 
  1. Показать все задачи
  2. Добавить задачу
  3. Отметить задачу выполненной
  4. Удалить задачу
  5. Статистика
  0. Выход
 
Нужно ввести номер нужного пункта и нажать Enter.
 
Добавление задачи:
  - Введите название задачи
  - Выберите предмет из списка (введи номер)
  - Выберите приоритет (1 - Высокий,
    2 - Средний, 3 - Низкий)
  - Введите дедлайн в формате ДД.ММ.ГГГГ
    или просто нажмите Enter чтобы пропустить
 
Отметить выполненной / удалить:
  - Сначала откроется список задач
  - Посмотрите ID нужной задачи (первая колонка)
  - Введите этот номер
